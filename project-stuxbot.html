SIEM Investigation

Hunting for Stuxbot

A threat-hunting case study focused on suspicious file staging, registry persistence, and PowerShell Remoting activity.

ElasticKibanaSysmonKQLMITRE ATT&CK

Project Overview

This investigation examines activity associated with a simulated Stuxbot campaign using endpoint and PowerShell telemetry in Elastic.

Investigation Goals

  • Identify tools written to C:\Users\Public.
  • Detect registry Run key persistence.
  • Trace PowerShell Remoting activity used for lateral movement.

Methodology

I reviewed Windows, Sysmon, and PowerShell logs, built focused KQL queries, correlated events by host and timestamp, and mapped confirmed behavior to MITRE ATT&CK.

Key Findings

Suspicious activity showed a clear progression from tool staging to persistence and lateral movement.

Response Recommendations

  • Isolate affected hosts.
  • Remove malicious persistence entries.
  • Reset compromised credentials.
  • Restrict and monitor PowerShell Remoting.
