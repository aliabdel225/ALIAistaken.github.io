SOC Home Lab with ELK Stack

A virtual security operations environment built to collect telemetry, simulate attacks, and investigate events.

ElasticsearchLogstashKibanaFilebeatSysmon

Project Overview

This project demonstrates the design and operation of a small SOC lab using Windows telemetry and the ELK Stack.

Architecture

  • Windows endpoint with Sysmon.
  • Linux-based ELK server.
  • Filebeat for log forwarding.
  • Kibana for searching and visualization.

Lab Workflow

I generated test activity, verified log ingestion, created searches and dashboards, and investigated suspicious events.

Skills Demonstrated

Log onboarding, SIEM troubleshooting, query development, event correlation, and incident documentation.

Lessons Learned

Reliable telemetry, accurate timestamps, and structured queries are essential for effective investigation.
