Packet Analysis

Network Scanning & Wireshark Analysis

A hands-on network investigation showing how Nmap reconnaissance appears at the packet level.

WiresharkNmapTCP/IPLinuxMITRE T1595

Project Overview

This lab used an attacker machine at 10.0.2.15 and a victim capture host at 10.0.2.3.

Objectives

  • Capture live traffic.
  • Identify SYN scanning behavior.
  • Analyze source and destination addresses.
  • Apply Wireshark display filters.

Analysis Process

I filtered TCP traffic, inspected SYN flags, reviewed port activity, and reconstructed the scanning pattern between the two systems.

Key Findings

The repeated SYN packets sent across multiple destination ports were consistent with network reconnaissance.

MITRE Mapping

The observed behavior maps to T1595 — Active Scanning.
